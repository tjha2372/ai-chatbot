# AI Chatbot

An AI-powered chatbot that responds to user queries in real-time using API integration. The application provides a smooth and interactive chat experience with dynamic responses.

## 🚀 Tech Stack
- React.js
- JavaScript
- REST APIs

## ✨ Features
- Real-time chat interface
- Dynamic responses using AI APIs
- Smooth user interaction without page reload
- Clean and responsive UI

## 📌 How It Works
- User types a message in the chat interface  
- The message is sent to an API  
- The API processes the request and returns a response  
- The response is displayed instantly on the screen  

## 🎯 What I Learned
- API integration in frontend applications  
- Handling asynchronous operations using async/await  
- Building interactive and user-friendly UI  

## 🔮 Future Improvements
